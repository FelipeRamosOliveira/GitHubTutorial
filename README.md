## 1° Parte (Criar)

```
git init
git add .
git add .\README.md
git status
git commit -m "teste"
git branch -M "main"
git remote add origin https://github.com/FelipeRamosOliveira/GitHubTutorial.git
git push -u origin main
```

## 2° Parte (Alterar)

```
git add .
git commit -m "teste"
git push origin main
```

## 3° Parte (Versionar)

```
git checkout -b "nova-funcao"
git add .
git commit -m "nova função"
git push origin nova-funcao
```

## 4° Parte (Fundir)

```
git checkout main
```
